# Node 工具包合集

## 多媒体/音视频

### 图片

- **[sharp](https://github.com/lovell/sharp)**：图片大小调整、旋转、提取、合成和伽马校正等操作；比 GraphicsMagick 和 ImageMagick 快四五倍。
- **[jimp](https://github.com/oliver-moran/jimp)**：纯 JS 实现的 Node 图像处理库，零原生依赖。
- **[gm](https://github.com/aheckmann/gm)**： GraphicsMagick 和 ImageMagick 的封装
- **[lwip](https://github.com/EyalAr/lwip)**：轻量的 Node 图片处理器
- **[node-qrcode](https://github.com/soldair/node-qrcode)**：二维码和条形码生成器
- **[qr-image](https://github.com/alexeyten/qr-image)**：二维码生成器
- **[jsQR](https://github.com/cozmo/jsQR)**：JS 实现的二维码识别器
- **[image-type](https://github.com/sindresorhus/image-type)**：检测 Buffer / Uint8Array 的图像类型

## 工程

### 打包工具

- **[webpack](https://webpack.js.org/)**：浏览器模块打包工具
- **[parcel](https://parceljs.org/)**：零配置 web 应用构建工具
- **[esbuild](https://esbuild.github.io/)**：极快的打包压缩工具，不使用 AST
- **[rollup](https://rollupjs.org/guide/en/)**：模块打包工具，常用于构建工具包
- **[grunt](https://gruntjs.com/)**：JS 任务执行器
- **[gulp](https://gulpjs.com/)**：流式构建系统，常用于 node 代码构建
- **[snowpack](https://www.snowpack.dev/)**： 是一个相对轻量的 bundless 方案
- **[vite](https://vitejs.dev/)**：新一代前端构建工具

## 开发

### 算法

- **[lru-cache](https://github.com/isaacs/node-lru-cache)**：LRU 缓存实现
- **[node-cache](https://github.com/node-cache/node-cache)**：node 内存缓存模块

### 网络/服务

- **[http-server](https://github.com/http-party/http-server)**：零配置起一个 http 服务器
- **[anywhere](https://github.com/JacksonTian/anywhere)**：零配置起一个静态资源服务器
- **[json-server](https://github.com/typicode/json-server)**：极速起一个 Rest API
