# ETF

# 1. ETF 基本知识

  • ETF分类：股票型ETF、债券型ETF、商品ETF、货币ETF 和 跨境ETF。

  • 场内交易：通过证券交易所进行的买卖活动

  • ETF链接基金：以绝大部分基金资产投资于目标ETF基金的基金产品，投资于ETF的比例不低于基金资产净值的90%。

# 2 指数介绍

指数的三个作用

  1. 市场的标尺，反映市场的平均收益和波动情况
  2. 指数基金的跟踪标的，按照指数的样本权重进行指数化投资
  3. 衍生产品的基础，股票指数期货、期权等衍生产品

# 2.1 指数研究方法

  1. 了解对应指数的编制信息
    • 信息查找
      • 中证指数公司编制的指数，在中证指数公司的官网找编制方案
      • 深圳证券信息公司编制的指数，在国证指数官网找对应编制方案
    • 三个信息
      1. 选样的方法
      2. 市值加权的方法
      3. 调仓频率和数量
  2. 分析市值、行业和十大权重股
  3. 了解指数的历史走势
  4. 指数的价值分析
    • 静态市盈率：取得是上一年度净利润
    • 滚动市盈率：取最近4个季度财务报表的净利润
    • 动态市盈率：预估下一个季度的净利润

# 2.2 常见A股宽基指数

  1. 上证50
    • 上证股票中规模大、流动性好50只
  2. 沪深300
  3. 中证100
    • 沪深300中规模最大的100只组成
  4. 中证500
    • 全部A股中剔除沪深300指数成分股及总市值排名前300的股票
  5. 创业板指数
    • 创业板最具代表的100家

# 2.3 常见A股行业指数

同属于某个行业的股票所组成的指数，具体而言，共有10个一级行业和27个二级行业

  1. 医药行业
    • 中证全指医药卫生指数（000991）：流动性和市场代表较好的股票
    • 沪深300医药指数（000913）：沪深300中的医药卫生股
    • 中证医药卫生指数（000933）：中证800中的医药卫生股
  2. 消费行业
    • 中证全指主要消费指数（000990）
    • 中证主要消费指数（000932）：中证800指数中的主要消费股
    • 沪深300主要消费指数（000912）
    • 上证主要消费指数（000036）
    • 中证全指可选消费指数(000989)
    • 沪深300可选消费指数（000911）
  3. 信息技术行业
    • 中证全指信息技术指数（000993）
  4. 军工行业
    • 中证军工主题指数（399967）

# 3. 股票型ETF

# 3.1 分类

  • 规模ETF：沪深300、中证500、创业板等等
  • 风格EFT：价值ETF、成长ETF
  • 行业ETF：医药ETF、金融地产ETF、信息技术ETF、传媒ETF
  • 主题ETF：环保ETF、京津冀ETF
  • 策略ETF：红利ETF
