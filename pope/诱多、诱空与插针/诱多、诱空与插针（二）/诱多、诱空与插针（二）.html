# 诱多、诱空与插针（二）

上一篇讲了插针 K 的含义，这一篇主要讲下短线交易中接针的操作。

在波动率较低的横盘震荡中，接针超短是主要的盈利方式之一，小仓位高杠杆往往能够几分钟内翻倍，当然，如果接了结果发现不是针，也可能直接爆仓。所以，在想要做超短接针操作之前，一定要知道有哪些情况不适合接针：

# 不适合接针

# 1. 持仓量高位时期

可以看 1 小时或者 4 小时 K 线的持仓量情况，如果主力已经建仓建的差不多了，持仓量在相对高位，随时可能发展出一轮新的趋势行情，那就不要再去想着做超短接针操作了。

# 2. 盘整震荡了相当长一段时间之后

与第一点类似，盘整震荡末期快出方向的时候，虽然也经常插针，但是接针的潜在风险已经大于潜在收益，就不适合再去接针了。至于多长时间算是盘整了相当长时间，这个需要结合走势进行判断，比较简单的判断办法是，参考上一段趋势启动之前盘整震荡的时间【图一】。接近过超过上一段盘整时间，就不要再去做超短接针操作了。

【图一】

# 3. 波动率较大的强趋势走势中

比如说去年年底 6000 长时间盘整后下破去 3000 的过程中，或者今年上半年 4200 大阳线上破之后的一段时间，强趋势中波动率极大，不适合高杠杆操作，尤其不适合去接逆趋势的针。比如 4200 大阳线之后，可以在盘整阶段寻找向下插针的时候接多，但一定不要在向上插针的时候接空，趋势走势是可以不断吃针的，就算接到好位置盈利空间也往往不是很大。

# 非常适合接针：

# 1. 盘整中震荡上行与震荡下行波段

在确认剧烈波动告一段落后，走势进入盘整期，波动降低，上涨和下跌常以震荡的形式出现，非常适合超短接针操作【图二】。震荡下行中经常出现向上插针，经常是主力为了打损队友盘，减轻车重故意为之，典型特点就是拉起后持仓量断崖式下跌。同理，震荡上行中也经常出现向下插针。

【图二】

如何判断走势接近结束有很多种方法，比如说波浪走了五浪，出现典型的 K 线形态，以及走势出现背离，上涨/下跌斜率渐缓。比如说【图三】的例子。

# 3. 走势接近一处易于识别的阻力位/支撑位时

币圈大多数人都会使用一些技术分析方法判断支撑和阻力，有些阻力位或支撑位是可以通过不同方法得到的，共识很强，第一次打到这样的阻力位或支撑的时候，很容易形成插针。最常见的就是整千整五百位置，比如 7000，7500，8000 这样的位置，第一次击穿往往是假突破，容易形成插针【图四】

# 注意事项：

# 1. 要有充足的盯盘时间和操作时间

插针行情的发生和结束往往就是一瞬间的事情，没时间盯盘的话不建议操作。

# 2. 接针前一定要有止损方案

一般止损可以选择下一个明显的支撑/阻力，如果没有这种位置，可以设置一个点或者两个点止损距离，止损单一定要在接单之后立刻挂上。如果接到之后马上收针出现盈利，可以挂针尖止损。

# 3. 见好就收

有很多时候插针的出现仅仅是主力的一次试盘动作，后续走势会继续向插针的方向运行。做超短的要诀就是不能贪，找确定性的机会，获得盈利，落袋为安。如果想要吃到完整的波段，可以使用另一个账户布局操作。
