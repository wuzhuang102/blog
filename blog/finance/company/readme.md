# 公司分析思路

## 1. 公司分析的基本思路
### 1.1 了解行业
阅读年报前，务必了解行业，部分公司会在年报中介绍自己的盈利模式。
- **市场特征**：自由竞争/垄断竞争/寡头竞争/完全垄断：这些决定行业内部玩家的定价策略、成本控制策略、市场分割特征。
- **是否为周期性行业**：一般认为有色、钢铁、建筑材料、房地产、金融等是周期性；白酒、农业、餐饮、食品饮料等是非周期性行业。周期性行业业绩下滑可能只是一个探底过程，但非周期行业下滑就可能是真的不行了。
- **行业阶段**：新兴行业/增长行业/成熟行业/衰退行业：有助于我们去分析公司的各类指标，特别是增长能力。
- **对资产的依赖程度**：重资产/轻资产：重资产行业人效作用不大，轻资产折旧意义不大
- **行业驱动力**：政策驱动/产品驱动/销售驱动/运营驱动/资本驱动：定位需要关注的财务数据指标
- **其它一些因素**：地域特征/客户分层/产品波动性
### 1.2 了解上市公司的基本商业模式
商业模式决定后期我们在阅读财务分析+财报的时候，关注哪些财务指标
#### 1.2.1 线性公司
公司为客户提供价值，直接从中获益
- To C
    - 什么产品
        - 实物/虚拟（边际成本）
        - 高单价/低单价（消费决策机制）
        - 奢侈品/必需品（消费弹性）
        - 高毛利/低毛利（盈利方式）
        - 高频/低频（消费节奏）
    - 客户群体特征
        - 是否有地区性（产品的扩展空间）
        - 年龄
        - 支付能力
    - 解决了消费者的什么需求
        - 基本需求/高级享受
        - 个人消费需求/家庭消费需求/送礼需求/社交需求
    - 怎么销售的
        - 渠道（线上、线下、合作、自营）
        - 营销方式（裂变、打折、代言）
        - 品牌
- To B
    - 什么产品
        - 实物/虚拟（边际成本）
        - 大额/小额（采购决策机制）
        - 长期/一次性（收入模型）
        - 过程产品/终端产品（分析财务指标）
        - 高毛利/低毛利（盈利方式）
        - 高频/低频（消费节奏）
        - （公司通过什么方式、在多长时间、以多大的代价、卖给怎样决策流程的、什么客户、满足了什么需求）
    - 产业链的什么位置
        - 上中下游
        - 市场地位（垄断、头部）
        - 多行业通用/垂直行业专用
        - 其它有助于分析公司的 议价能力、定价能力、价格销量的因素
    - 客户特征
        - 大B/小B
        - G
        - 在客户采购中的地位（主营业务、通用业务、边缘但必须业务）
        - 客户决策模式（谁出钱、谁获益、谁拍板）
        - 其它有助于理解销售采购对象采购细节的要素
#### 1.2.2 平台型公司
为买卖双方或多方提供交易的产所，从撮合中获益
- 平台价值
    - 技术驱动/资源驱动/规模驱动
    - 平台对谁收费/怎么收费/哪类资源收费
- 平台的参与方
    - B/C/G
    - 分别有什么特点
    - 平台满足用户的什么需求
    - 用户粘度、付费意愿、续费意愿
#### 1.2.3 混业经营公司
- 是否能交叉销售
- 是否能平滑业绩
- 是否能财务协同（财务互补？）
### 1.3 业务总览
年报最开始的部分是公司管理层对集团业务、战略、市场和重点数据的解读
#### 1.3.1 根据对行业的和公司的理解 check 管理层言论
- 公司对行业的分析与我们自己的是否一致
- 公司对市场的展望与我们是否一致
- 公司战略是否有重大调整或新业务线：出于什么原因（政策、行业发展、经济周期）
- 公司提示的风险：上市公司喜欢把所有的风险都列出来
- 组织架构调整：意味着对不同业务线重视程度的变化、对不同产品线资源的投入会变化
#### 1.3.2 查看对本本公司重要的指标
业务讨论部分给出的指标都是很重要的<br>
第二步中事先准备一个重要数据表格，年报中没有的部分，就去其它渠道找
- 比如SAAA行业：现有付费客户数、客单价、续费率
- 游戏公司：旗舰游戏需要单独简历模型
- 有色行业：需要把市场价格在报告期的走势中去看
- 互联网平台：UV、PV、用户市场数据

之后就需要阅读业务数据与财务数据，**核心工作就是 证实/证伪 公司的说法**，所有分析，同比、环比、行业平均都是必要操作

- 异常数据：事出反常必有妖
- 利润表好操作、资产负债表稍微难点，现金流量表最难操作。数据冲突时我们习惯看现金流量表（和银行打交道很难），它可以佐证我们对企业商业模式和经营策略的态度
    - 经营活动现金流 -- 是否赚钱
    - 投资活动现金流 -- 投资再生产的效果及态度
    - 筹资活动现金流 -- 公司对股东、债主资金的态度
### 1.4 盈利能力
#### 1.4.1 各项盈利指标
- 销售毛利率
- 销售净利率
- OCF 、 FCF
- EBIT/EBITDA 等

选什么指标要有行业积累：不同公司，主流评估盈利能力的指标是不一样的，行业充分竞争的市场里，成本的固定百分百就是终端价，这时毛利率就没什么用；其它行业，内部折旧政策千差万别 EBITDA 就好一点
#### 1.4.2 和资产相关的各类盈利指标
- 总资产回报率
- 净资产回报率
### 1.5 运营能力
#### 1.5.1 企业内部各种周转率
核心目的是评价企业内部，有效利用资金、资源，运用资产的能力
- 存货周转率
- 固定资产周转率
- 现金循环周期
#### 1.5.2 与上下游的各类周转率
- 应收（率、天数）
- 应付
应收应付的坑可以查一下相关分析思路
#### 1.5.3 各类费效比
- 人效比
- 营销费用比
- 管理费用比
### 1.6 财务能力
财务能力主要分析爱议价公司在资本市场上的能力：但不同的行业也有不同，例如地产与游戏
#### 1.6.1 融资能力
- 增发能力：第一判断公司是否通过增发稀释现有股东股权，防止被收割；第二增发如果是为了企业更好的发展，要了解公司在股权资本市场上融资的能力
- 举债能力：关注周期、成本、各年度到期情况
- 其它融资类型：贷款、金融租赁、可转债、供应链金融等的周期、成本、到期情况
#### 1.6.2 投资能力
上市公司是一个金融平台，不仅是用来融资的，还是一个投资平台

很多公司做产业投资对财务性回报要求不高，但对产业协同极为重视
- 了解重点被投企业的状况
- 了解投资回报情况
- 分析被投企业、子公司、兼并的业务协同
#### 1.6.3 财务健康度的比率
- 资产负债率
- 流动比率
- 速动比率
- 现金流量比率
- 偿债保障比率
### 1.7 年报中的重要事项
年报中对公司经营有显著影响的条目，都会进行针对性的说明
- 重要研发（每个行业都有自己的重要研发，需要我们的行业知识）
- 重要新增资产（土地储备、制造业的新厂新产线、有色行业的矿）
- 重要人员变动
- 一些会计层面的重要变动
### 1.8 增长能力
#### 1.8.1 数量分析
- 有色行业、地产行业做收入线性预测
- 零售行业用环比
- 垄断性行业用竞争市场的模型
#### 1.8.2 行业驱动型
分析一家企业一定要拆分增长来自行业还是来自企业
- 政策、宏观因素：例如电动车
- 行业阶段：不同发展阶段，行业对公司的影响不同
- 周期性行业的波动：有色需要看商品、基建需要看财政政策，化工看原油
- 非周期行业的影响因素：视频的就要看市场偏好、医药就看医疗改革
