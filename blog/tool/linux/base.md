# Linux基本知识

## 1. Linux 文件目录
```
    | -- bin/       二进制可执行文件，常用命令一般在这
    | -- sbin/      二进制可执行文件，只有root才能访问
    | -- etc/       系统管理和配置文件
    |   | -- sysconfig/
    |       | -- network-scripts/
    | -- home/      用户目录基点
    | -- root/      超级用户主目录
    | -- usr/       用户级的一些文件，里面的目录结构类似linux根目录
    |   | -- bin/       多数日常库(git、yum)
    |   | -- local/     个人安装的软件；与`/usr`目录结构相似
    |   | -- sbin/      超级用户不需要的系统管理程序
    |   | -- lib/
    | -- lib(64)/   系统最基本的共享链接库和内核模块
    | -- boot/      系统引导时的各种文件，linux内核文件
    | -- opt/       额外可选应用程序包所放置的位置，例如tomcat等
    | -- proc/      虚拟文件目录，存放系统内存映射，可直接访问这个目录来获取系统信息
    |   | -- cpuinfo    处理器的相关信息(cpu cores、processor、siblings、physical id)
    |   | -- version    系统内核版本号(与uname -a命令作用类似)
    |   | -- <pid>/     进程文件目录
    |       | -- cmdline    启动当前进程的完成目录
    |       | -- cwd/       当前进程工作目录软链  
    |       | -- environ    当前进程环境变量列表
    |       | -- exe        启动当前进程的可执行文件的软链
    |       | -- fd         保存当前进程持有的文件描述符
    |       | -- limit      当前进程使用资源的软限制、硬限制
    |       | -- task/       保存当前所运行的每一个线程的相关信息 与/proc/<pid>目录结构相似
    | -- var/       运行时需要改变的数据文件，大文件的溢出区，比如各种服务的日志文件
    |   | -- log/      各程序的日志目录
    | -- mnt/       挂载移动设备
    | -- srv/       程序
    | -- dev/       设备文件
    | -- media/     
    | -- tmp/       零时文件
    | -- lost+found 一般是空的，系统非正常关机时留下的文件
```
