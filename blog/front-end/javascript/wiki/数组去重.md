# 数组去重

## indexOf 去重

``` js
function unique(arr) {
    var arr_res = []
    for(var i = 0; i < arr.length; i ++) {
        if(arr_res.indexOf(arr[i]) === -1) {
            arr_res.push(arr[i])
        }
    }
    return arr_res
}
```


## 排序后去重
js提供的排序方法排序结果不总是正确的，它的原理是把数据元素转换成字符串再进行排序的，实际生产中还是以自己封装的方法为准
``` js
function unique(arr) {
    var sort_arr = arr.concat().sort(), arr_res = []
    for (var i = 0; i < sort_arr.length; i++) {
        if(!i || sort_arr[i] !== sort_arr[i-1]) {
            arr_res.push(sort_arr[i])
        }
    }
    return arr_res
}
```

## 明确数组是否已经排序下的去重
``` js
function unique(arr, is_sorted = false) {
    var arr_res = []
    if (is_sorted) {
        for (var i = 0; i < arr.length; i++) {
            if (!i || arr[i] !== arr[i]) {
                arr_res.push(arr[i])
            }
        }
    } else {
        for (var i = 0; i < arr.length; i++) {
            if (arr_res.indexOf(arr[i]) === -1) {
                arr_res.push(arr[i])
            }
        }
    }
}
```
