Node 工具包合集

多媒体/音视频​

图片​

  • sharp：图片大小调整、旋转、提取、合成和伽马校正等操作；比 GraphicsMagick 和 ImageMagick 快四五倍。
  • jimp：纯 JS 实现的 Node 图像处理库，零原生依赖。
  • gm： GraphicsMagick 和 ImageMagick 的封装
  • lwip：轻量的 Node 图片处理器
  • node-qrcode：二维码和条形码生成器
  • qr-image：二维码生成器
  • jsQR：JS 实现的二维码识别器
  • image-type：检测 Buffer / Uint8Array 的图像类型

工程​

打包工具​

  • webpack：浏览器模块打包工具
  • parcel：零配置 web 应用构建工具
  • esbuild：极快的打包压缩工具，不使用 AST
  • rollup：模块打包工具，常用于构建工具包
  • grunt：JS 任务执行器
  • gulp：流式构建系统，常用于 node 代码构建
  • snowpack： 是一个相对轻量的 bundless 方案
  • vite：新一代前端构建工具

开发​

算法​

  • lru-cache：LRU 缓存实现
  • node-cache：node 内存缓存模块

网络/服务​

  • http-server：零配置起一个 http 服务器
  • anywhere：零配置起一个静态资源服务器
  • json-server：极速起一个 Rest API
